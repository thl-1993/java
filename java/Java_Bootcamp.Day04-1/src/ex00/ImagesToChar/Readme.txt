инструкции по компиляции и запуску вашего исходного кода из консоли (не IDE).
Инструкция написана для состояния, когда консоль открыта в корневой папке проекта.

Необходимо находиться в папке src/ex00/ImagesToChar (корневая папка текущего проекта)
$ cd src/ex00/ImagesToChar/

либо компилировать по одному файлу, но в порядке зависимостей (неудобно)
$ javac -classpath ./target -d ./target  src/java/edu/school21/printer/logic/Reader.java
$ javac -classpath ./target -d ./target  src/java/edu/school21/printer/app/Program.java

/*
javac — это Java-компилятор, который компилирует исходные файлы .java в байт-код .class.
-classpath ./target — указывает компилятору, где искать скомпилированные файлы классов и библиотеки для разрешения зависимостей.
./target — это директория, где будут находиться уже скомпилированные классы (если есть зависимости между файлами, они будут подтянуты отсюда).
-d ./target — указывает, куда сохранять скомпилированные файлы. Все скомпилированные .class файлы будут помещены в папку ./target.
src/java/... — это путь к файлу исходного кода, который нужно скомпилировать (например, Reader.java и Program.java).
*/

Компиляция всей папки:
$ javac -sourcepath ./src/java -classpath ./target -d ./target src/java/edu/school21/printer/app/Program.java

/*
Эта команда компилирует всю папку src/java разом:

-sourcepath ./src/java — указывает корневую папку исходников. Компилятор будет искать все файлы .java в этой папке.
-classpath ./target — директория для поиска зависимостей (скомпилированных классов).
-d ./target — директория, куда компилятор поместит все скомпилированные файлы.
src/java/edu/school21/printer/app/Program.java — основной файл для компиляции. Зависимости от других файлов подтянутся автоматически благодаря параметрам sourcepath и classpath.
*/

папка корня проекта:
    -sourcepath ./src/java
место хранения уже скомпилированных исходников, для поиска зависимостей:
    -classpath ./target
место для складывания при компиляции:
    -d ./target
файлы для компиляции, зависимости подтянутся:
    src/java/edu/school21/printer/app/Program.java

Запуск
$ java -classpath target/ edu.school21.printer.app.Main it.bmp

/*
java — это команда для запуска Java-программ.
-classpath target/ — указывает путь к скомпилированным .class файлам (они находятся в папке target, куда мы их скомпилировали).
edu.school21.printer.app.Main — полное имя класса, который содержит метод main, и с которого начинается выполнение программы. Это квалифицированное имя класса (package.className).
В моём случае, класс Program находится в пакете edu.school21.printer.app.
it.bmp — это аргумент командной строки, который передается в программу. В данном случае, это путь к файлу it.bmp, который будет обработан программой.
*/
